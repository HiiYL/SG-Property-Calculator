# Singapore Property Calculator

A comprehensive property investment calculator for Singapore real estate, helping buyers analyze costs, returns, and affordability.

## Features

- **Stamp Duty Calculations**: BSD (Buyer's Stamp Duty) and ABSD (Additional Buyer's Stamp Duty) based on latest IRAS rates
- **Loan Analysis**: Monthly payments, TDSR (Total Debt Servicing Ratio) checks, and amortization
- **Investment Returns**: ROI, rental yield, and appreciation projections
- **Affordability Assessment**: Cash/CPF requirements and eligibility checks
- **Tax Considerations**: Rental income tax and SSD (Seller's Stamp Duty) calculations
- **Supports**: Singapore Citizens, PRs, and Foreigners across HDB and Private properties

## Tech Stack

- **React 19** with TypeScript
- **Vite 7** for fast development and builds
- **TailwindCSS 4** for styling
- **Lucide React** for icons

## Getting Started

### Prerequisites

- Node.js 18+ 
- npm or yarn

### Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/sg-property-calculator.git
cd sg-property-calculator

# Install dependencies
npm install

# Start development server
npm run dev
```

### Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build locally
- `npm run lint` - Run ESLint
